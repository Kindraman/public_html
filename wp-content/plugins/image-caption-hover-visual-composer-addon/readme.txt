== Description ==

A simple solution to add images in your site with modern hover effects including a bunch of custom options and settings.

== Installation ==
1. Go to plugins in your dashboard and select \'add new\'
2. Search for \'Image Caption Hover\', install it and activate
3. It will appear in visual composer Contents section
